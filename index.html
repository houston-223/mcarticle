May 2009

The Birth of Minecraft

Markus "Notch" Persson releases the first public version of Minecraft. Originally called "Cave Game", this early version featured basic block placement and mining mechanics. The game was inspired by games like Dwarf Fortress and Infiniminer.

December 2010

Beta Release & Growing Popularity

Minecraft enters beta phase, adding features like biomes, beds, and new block types. The game gains significant traction with over 100,000 purchases in the first weekend of beta release. Multiplayer support is introduced, setting the foundation for the massive online communities to come.

November 2011

Official Release & The Ender Dragon

Minecraft 1.0 launches with the addition of The End dimension and the Ender Dragon boss fight. This official release version also introduces brewing, enchanting, and experience points. The game has now sold over 4 million copies.

September 2014

Microsoft Acquisition

Microsoft acquires Mojang for $2.5 billion. Markus Persson steps away from Minecraft development. The acquisition leads to expansion onto new platforms including PlayStation, Nintendo Switch, and mobile devices.

July 2016

Minecraft: Education Edition

Microsoft releases Minecraft: Education Edition, designed for classroom use. This version includes features tailored for educators, allowing teachers to use Minecraft as an educational tool in subjects ranging from history to computer programming.

September 2017

Better Together Update

The Better Together Update unifies Minecraft versions across multiple platforms, enabling cross-platform play between Xbox, Switch, mobile, and Windows 10. This update also introduces the Marketplace for community-created content.

June 2020

Nether Update

The massive Nether Update completely revamps the Nether dimension, adding new biomes, mobs (piglin, hoglin), structures (bastions), and blocks. This update represents one of the most significant content expansions since the game's release.

Continued Evolution
